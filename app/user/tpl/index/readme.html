使用说明

充值:

先进入首页的 充值页面

使用手机微信扫描二维码,切记!

备注中输入你的用户名

否则无法自动到账,将会人工退款(最好反馈一下,如果没看见钱就归我了)


开通服务:

进入开通页面,将会有套餐供选择

选择合适自己的,点击确定开通


使用客户端:

Windows 用户:

先在首页点击下载客户端

直接打开客户端 登陆你的账号

点击X是无法关闭的,客户端会自动缩小到右下角,右键点击图标退出


Android 用户:

先在首页点击下载客户端

安装完毕后,打开

其他用户:

不好意思,我们没有其他用户,请自行去网上搜索相关教程233(有人提供教程或者设备的话我会上传的,开发者并没有IOS,win phone等设备)

返回首页
